## Описание проекта
Данный проект представляет собой 3D-симуляцию физического принципа рычага, разработанную на игровом движке **Unity 6**. Симуляция наглядно демонстрирует принцип рычага. Красный куб весит 20 кг, а синий 8 кг. Данный принцип заключается в том, что синий куб находится на длинном плече, а 
красный на коротком. Рычаг способен преобразовывать силу так чтобы при малом усилии синего куба можно было поднять красный, более тяжелый куб.
